HTML Tags
Tag Name Description
<title> Title tag Contains the title of the webpage
<header> Header tag Contains the header element of the page
<head> Head tag Contains metadata required for the page
<h1> , <h2> , <h3>,... ,<h6> Heading tags Used for headings, with <h1> being main and <h6> being least subheading
<body> Body Tag Contains all the main information of the webpage
<footer> Footer tag Contains information meant for the footer of the page
